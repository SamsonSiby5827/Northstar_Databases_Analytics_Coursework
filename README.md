 NorthStar Databases and Analytics Coursework

This repository contains the Google Colab notebooks, datasets, outputs and screenshots used for the Databases and Analytics coursework based on the NorthStar Urban Mobility and Logistics case study.

The coursework analyses NorthStar's operational data to identify issues related to delivery delays, failed services, customer complaints, hub performance, route efficiency, app events and data fragmentation. The work uses SQL within R, R analytics, Python data processing, MongoDB Atlas, PyMongo and query optimisation.
